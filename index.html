Built on Base: 100 Ecosystem Facts

  1. Base is an Ethereum Layer-2 blockchain developed by Coinbase to enable faster, cheaper, and more accessible on-chain transactions while leveraging Ethereum’s security.[1]
  2. Base is built using the OP Stack, an open-source Ethereum Layer-2 framework originally developed by Optimism.[2]
  3. The Base testnet was publicly announced by Coinbase in February 2023.[3]
  4. Base’s goal is to onboard the next billion users to the web3 ecosystem by making blockchain easier to use.[3]
  5. Coinbase officially launched the Base mainnet to the public on August 9, 2023 at the start of the first Onchain Summer.[4]
  6. The initial launch included 100+ decentralized apps and service providers on Base.[4]
  7. Base was incubated within Coinbase Global, Inc., the leading regulated crypto exchange founded by Brian Armstrong and Fred Ehrsam.[5]
  8. Base did not launch with a native token and has publicly stated it has no current plans to issue one.[3]
  9. Base uses optimistic rollups to bundle transactions off-chain and post validity proofs back to Ethereum, increasing throughput and reducing fees.[2]
  10. Early Coinbase documentation defined Base as a secure, low-cost, developer-friendly Layer-2 for anyone to build decentralized applications.[3]
  11. Coinbase’s strategy with Base has been described as part of a broader “Superchain” vision of interoperable Ethereum Layer-2 networks.[2]
  12. Base records transactions at a fraction of Ethereum mainnet’s cost, often under $0.01 per transaction.[1]
  13. As of October 2025, Base’s total value locked (TVL) exceeded $5 billion, marking rapid ecosystem growth.[6]
  14. Base reached approximately 844,000 active addresses daily as of late 2025.[6]
  15. Base became one of the largest Ethereum Layer-2 networks, second only to Arbitrum by some measures.[6]
